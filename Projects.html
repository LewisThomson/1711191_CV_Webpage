Python Password Cracker

The following project was created in 2016 , the code was written in python 3.0. The password cracker uses a brute force attack in order to determine how many attempt and how long it would take to crack a password. If the software determains that it would take an indefinate amount of time to crack it will instead use a dictionary attack in attempt to crack the password.
